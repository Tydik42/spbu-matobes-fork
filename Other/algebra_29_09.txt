\begin{theorem} (Алгоритм Евклида)
    Для любых чисел a,b алгоритм Евклида заканчивается за конечное число шагов и его результат равен НОД (a, b).

\end{theorem}

\begin{lemma}
    $\forall$ a, b, k НОД(a, b) = НОД(a + kb, b)


\end{lemma}

\begin{proof}
    Докажем, что множество общих делителей совпадает.

    Пусть $M_1$ - множество общих делителей чисел a и b.

    и $M_2$ - множсетво общий делителей a+kb и b.

    Докажем что множества $M_1$ и $M_2$ совпадают.

    \begin{enumerate}
        \item Докажем, что $M_1 \subset M_2$
        $]d \in M_1 => a \vdots d, b \vdots d =>$

        $kb \vdots d => a + kb \vdots d => d$  - Общий делитель.

        $a + kb и b => d \in M_2$

        \item $M_2 \subset M_1$
        
        $]d \in M_2 => a + kb \vdots d, b \vdots d$

        $=> a = (a + kb) - kb \vdots d$

        $=> d \in M_1$
    \end{enumerate}
\end{proof}

\begin{proof} (теоремы)

    \begin{enumerate}
        \item Алгоритм заканчивается
        
        $a \geq b > r_1 > r_2 ...$, где $r_i$ - остаток

        Кол-во $r_i$ конечно.

        \item результат равен НОД(a, b)
        
        Покажем, что шаг алгоритма не меняет НОД

        Если $a \vdots b$, то НОД(a,b) = b

        Если a не делится на b, то итог алгоритма не меняе НОД.

        НОД(a, b) = НОД(a -bq, b)

        (a - bq : остаток)


    \end{enumerate}
    
\end{proof}

\section{Линейной представление НОД}

\begin{theorem} (линейное представление множеств)
    
    Пусть $a, b \in \N$, тогда

    \begin{enumerate}
        \item Существует $x, y \in \Z$ такие, что $a_x + b_y = (a, b)$
        \item Пусть k - общий делитель a и b, тогда $(a, b) \vdots k$
    \end{enumerate}

\end{theorem}

\begin{proof}
    Положим $M = \{au + br | u, v \in \Z\}$

    Обзначим через d наименьший положительный элемент M,

    через x, y - такие числа, что d = ax + by

    и докажем что d - общий  делитель a и b

    и если k - общий делитель a и b, то d делится на k.

    % pic 1

    \begin{enumerate}
        \item Докажем, что $a, b \vdots d$
        Пусть a не делится на d. Делим a на d с остатком.
        $a = dq + r, 0 < r < a$

        $r = a - dq = a - (ax + by)q = a(1 - qx)+ b(-qy) \in M$

        $0 < r < d$, противоречие

        $r \in M$

        Для b - аналогично.

        \item Проверяем, что d делится на любой общий делитель.
        
        Докажем, что если k - делитель a и b, то d $\vdots$ k

        d = ax + by
        $a \vdots k, b \vdots k$

        $a \vdots k =>$
        $b \vdots k =>$

        $=> ax + by \vdots k$


    \end{enumerate}
\end{proof}

\begin{remark}
    Линейное представление можно найти с помощью алгоритма Евклида


\end{remark}

\begin{remark}
    уравнение $ax + by = c$ имеет решение <=> $c \vdots (a, b)$
\end{remark}

\section{Взаимно простые числа}

\begin{definition}
    Числа a и b называются взаимно простыми, если их НОД (a, b) равен 1.
\end{definition}

\begin{definition}
    Числа $a_1, a_2 ... a_k$ называются взаимно простыми в совокупности, если их НОД равен 1.
\end{definition}

\begin{definition}
    Числа $a_1, a_2, ..., a_k$ Называются попарно взаимно простыми, если любые два из них взаимно простые.
\end{definition}

\begin{eg}
    6, 10, 15 - взаимно простые в совокупности, но не попарно взаимно простые.
\end{eg}

\begin{lemma}
    Числа a и b взаимно просты <=> когда существует x, y, такие что $a_x + b_y = 1$


\end{lemma}

\begin{proof}
    =>) по т.о. линейного представления НОД.
    <=) $\exists x, y$ ax + by = 1
    Пусть d = (a, b) и он $\neq 1$ 
    => тогда $ax + by \vdots d$, 1 не делится на d. Противоречие
\end{proof}

Свойство: (взаимная простота с произведением)

Если каждое из чисел $a_1, ..., a_k$ взаимно просто с b, то произведение $a_1 \cdot a_k$ тоже взаимно просто с b.

\begin{proof} (индукция)
    База k = 2

    Докажем, что если $a_1$ и $a_2$ взаимно просты с b, то $a_1$ и $a_2$ взаимно просты с b

    По л. $\exists x_1, y_1, x_2, y_2$, такие, что

    $a_1x_1 + by_1 = 1, a_2x_2 + by_2 = 1$

    Перемножим

    $(a_1 a_2)(x_1 x_2) + b(a_1 x_1 y_2 + y_1 a_2 x_2 + b y_1 y_2) = 1$
    
    Получили линейной представление через $a_1 a_2$ и b, значит они взаимно просты.

    
    Переход k -> k + 1

    $a_1, ..., a_k, a_{k + 1}$ - взаимно просты с b

    $a_1, ..., a_k$ - взаимно просты с b (индукионной предположение для K)=> значит их произведение взаимно просты с b

    $a_1 \cdot ... \cdot$ взаимно просты с b, $a_{k + 1}$ взаимно просто с b (индукционное предположение для 2)=>

    $(a_1 \cdot ... \cdot a_k) \ cdot a_{k + 1} $ взаимно просты с b


\end{proof}

Свойство (связь с делимостью)

\begin{enumerate}
    \item Пусть произведение ab $\vdots c$, a и c - взимно простые, тогда $b \vdots c$
    \item Пусть $a \vdots b, a \vdots v$, b и c - взаимно простые, тогда $a \vdots bc$
\end{enumerate}

\begin{proof}
    \begin{enumerate}
        \item $\exists x, y: ax + cy = 1$ Умножим на b
        Получим $(a, b)x + bcy = b$

        $ab \vdots c => ab x \vdots c$

        $b \subset y \vdots c$

        =>

        $b \vdots c$

        \item $a = bk, a = cm$, $\existsx,y : bx + cy = 1$
        Умножим на k

        Получим, что $k = bkx + cyk = ax + cyk = cmx + cyk \vdots c$

        $k = cz$, $a = bk = (bc)z \vdots bc$
        
        $(a_1 \vdots ... \vdots a_k) \vdots a_{k + 1} $ взаимно просто с b


    \end{enumerate}
\end{proof}

\section{Простые числа}

\begin{definition}
    Число p назвается простым, если p > 1  и у p нет натуральных делителей, кроме 1 и P.

    
\end{definition}

\begin{definition}
    Число n называется составным, если n > 1 и n не простое.
\end{definition}

\begin{notation}
    Множество простых чисел будем обозначать $\mathbb{P} $.
\end{notation}

Свойство: Число a составное <=} $\exists b, c$ такие, что
$a = bc, 1 < b, c < a$

\begin{proof}
    =>} докажем, что если $a \notin  \mathbb{P}$. У f есит делитель b: $b \neq 1, b \neq a$
    => 1 < b < a.
    
    $\exists c \vdots a = bc$
    $c = \frac{1}{b}$

    $\frac{a}{a} < c < \frac{a}{1}$
    a/a = 1, a/1 = a

    <=) a=bc, 1 < b < a => y a есть делитель $\neq 1, \neq a$
    => $a \notin \mathbb{P} $
\end{proof}

\begin{lemma}
    У любого натурального числа большего 1, есть хотя бы один простой делитель 
\end{lemma}

\begin{proof}
    Индукция.

    База n = 2, делитель 2

    Переход. Предположим, что n > 3 и \forall k : 1 < k < n,

    у k есть простые делители.

    Докажем, что у n есть простые делители.

    \begin{proof}
        Случай 1. $n \in \mathbb{P}$ n - простой делитель n.
        Случай 2. $n \notin \mathbb{P}$, тогда $ n = k \vdot m$, 1 < k, m < n
    \end{proof}

    По индукционному предположению, $\exists p \in P : k \vdots p$
    $=> n \vdots p$
\end{proof}

\begin{theorem}(теорема Евклида)
    Множество простых чисел бесконечно. 
\end{theorem}

\begin{proof}
    Пусть $p_1, ..., p_l$ - все простые числа.

    Положим $N = p_1 \vdots ... \vdots p_k + 1$. Тогда, по лемме у N есть некоторый простой делитель p.

    $N \vdots p_1. N rp_2 ... N$ не делится на $p_k$ т.к.

    если $N \vdots p_1$, то

    $1 = W - p_1(p_1 ... \vdots p_{i - 1} p_{i + 1} ... p_k) \vdots p_i$

    $p \in \mathbb{P} , p \neq p_1, ..., p_k$ Противоречие.
\end{proof}


\begin{theorem}(Дирихле)
    Если $(a, m) = 1$, тогда существует бесконечно много простых чисел вида $a + km$

    (без доказательство, оно сложное)
\end{theorem}

\section{Основная теорема арифметики}

\begin{theorem}(основная теорема арифметики)
    Любое натуральное число больше единицы, можно представить в виде произведения простых чисел.

    И такой предтавление единственно с точностью до порядка сомнож.
\end{theorem}

\begin{proof}
    \begin{enumerate}
        \item Индукция
        
        База n = 2,  2 = 2 - предположение

        Переход. Предположим, что все числа, меньшие n, раскладываются в произведения простых.

        Докажем для n. 

        Случай 1: n простое. n = n - разложение
        Случай 2: $n \notin \mathbb{P}, n \vdots P$.

        1<p<n

        $1 < \frac{n}{p} < n$

        По индукционному предположению, $\frac{n}{p}$ можно разложить, $\frac{n}{p} = p_1p_2 ... p_k$
        => n = $p \vdots p_1 \vdots ... \vdots p_k$ - разложение.
    \end{enumerate}
\end{proof}

Единств. Псть n - наименьшее число, которое можно разложить двумя способами .

$n = p_1 ... p_n $, $n = q_1 ... q_n$

Если $p_i = q_j$ для некоторых i, j, то $\frac{n}{p_i} = \frac{n}{1_j}$ тоже

раскладывается двумя способами, n не минимально.

Рассмотрим $p_1$
$q_1 \neq p_1$, $q_2 \neq p_1$, ..., $q_m \neq p_1$

$q_1$ взаимно просто с $p_1$, т.к. делители Q_1 - это 1 и $q_1$, $p_1 \percent q$,

$q_2$ взаимно просто с $p_1$

$q_m$ взаимно просто с $p_1$


По свойству взимно простых чисел,

$q_1, q_2 ..., q_m$ взимно просто с $p_1$.

$q_1, q_2, ..., q_m$ - Это разложения

$n = q_1 ... q_m$

$\vdots p_1$ взаимно просто с $p_1$ Противоречие.
